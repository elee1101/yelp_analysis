# 🍽️ Yelp Data Analysis (SQL Project)

## 📌 Overview
Analyzed Yelp dataset using PostgreSQL to extract insights on business performance, user behavior, and review patterns.

## 🎯 Objectives
- Build relational database from raw data  
- Perform SQL-based analysis (JOIN, aggregation, window functions)  
- Extract real-world insights  

## 🛠️ Tech Stack
PostgreSQL, SQL, Python (data cleaning)

## 🔍 Key Highlights
- Identified top-performing businesses  
- Analyzed user rating behavior  
- Compared stored vs actual ratings  
- Used window functions for advanced insights
